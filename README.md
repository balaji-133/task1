# Learnato Discussion Forum — In‑Memory MERN
A hackathon‑ready forum with a beautiful, responsive UI, real‑time updates, and **array‑only storage**.

## Run Locally (no Docker)
### Backend
```
cd backend
npm install
npm run dev
```
### Frontend
```
cd frontend
npm install
npm run dev
```
Open http://localhost:5173

## With Docker
```
docker-compose up --build
```
Frontend: http://localhost:5173
Backend:  http://localhost:4000/health
